# DiscordBot

This is a Discord Bot created with Node.js That you can use **YourSelf**.

All you need to do it's to fill the "config.json" file.

## config.json

```json
{
    "token": "YourBotToken",
    "prefix": "YourPrefix",
    "mongoPath": "YourMongoPath"
}
```
